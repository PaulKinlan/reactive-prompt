This demo shows the basics of how to use reactive-prompt with Gemini

Enter your Gemini API key and click the button to run a prompt and see the response.

Expected: Output "hello Paul", then 3 seconds later output "hello Serene".

const name = signal("Paul");
const config = new GeminiPromptConfiguration();
config.key = "YOUR_API_KEY";
const response = prompt`${config}Repeat the following words: "hello ${name}".`;

setTimeout(() => {
  name.value = "Serene";
}, 3000);

effect(async () => {
  console.log(await response.value);
});
