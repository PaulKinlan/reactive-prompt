This demo shows the basics of how to use reactive-prompt

Click the button to run a prompt and see the response.

Expected: Output "hello Paul", then 500ms later output "hello Serene".

const name = signal("Paul");
const response = prompt`Say "hello ${name}".`;

setTimeout(() => {
  name.value = "Serene";
}, 500);

effect(() => {
  console.log(response.value);
});
